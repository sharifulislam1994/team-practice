# For new branch

```bash
git checkout -b [branch name]
```

# For see branches how many branches

```bash
git branch
```

# For switch before branch

```bash
git checkout -
```

# For switch branch

```bash
git checkout [branch name]
```

# For delete branch

```bash
git checkout -d [branch name]
```

# For remote pull

```bash
git pull origin [branch name]
```

# For git clone

```bash
git clone [url]
```
